LogMergeTool v38 UI Usability

反映内容:
- Output FolderをSource Folderと同じにする「Same as Source」ボタン追加
- Log Typesに小型のAll / Noneボタン追加
- Date Rangeはカレンダー入力継続
- Progress popupを大きく、文字を大きく、柔らかいデザインに変更
- Log Viewerに表示ペイン選択チェックボックス追加
  - 1/2/3/4ログ表示に加えて、どのペインを表示するかを選択可能
- Log Viewerのデフォルト表示列をTimestamp / Message / Levelに整理
  - File, SourceType, Line, CategoryなどはColumns...から必要時に表示
- 各ペインにColumns...ボタン追加
  - チェックボックス形式で列の表示/非表示を切替
- Timestampは固定幅、Messageはウィンドウ幅に合わせて広がる設定
- マウスオーバーの値ポップアップは継続

注意:
- 既存のPlugin Manager / Update Tool / Smart File Discovery等はv37から継続しています。
