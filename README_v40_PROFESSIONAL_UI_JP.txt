LogMergeTool v40 Professional UI 評価版

主な反映内容:
- Main画面をSTART中心の操作導線へ整理
- Mergeは START、Importは Import ボタンへ統合
- Updateは Update... ボタンへ統合し、manifest.jsonから File Type Plugin / Tool Update を自動判別
- Smart File Discoveryをカレンダー入力・リアルタイム反映型に変更
- Smart File Discoveryでファイル選択状態を保持
- Smart File Discoveryの開始ボタンを START に変更
- Detected Log Typesに All / Clear を維持
- Merge完了時の完了メッセージを出さず、自動でLog Viewerを前面表示
- Main / Viewer共通の大きめで柔らかいProgress Popupへ変更
- Log Viewerは初期表示サイズを画面約1/3に抑制
- Log ViewerのNoise FilterデフォルトOFF
- Log ViewerのLoad Visible Logsを LOAD LOGS 表記へ変更
- Log Viewerのタイトル行を廃止し、Logドロップダウン行を色付きに変更
- Message優先表示の列構成を維持
- WaterSystemをWSとは別ログタイプとして独立
- WaterSystemはMainState / CoolingStatus / Errorのみ取り込み

注意:
- v40はUI/UX刷新の評価版です。
- プラグイン方式・Core Update方式はv37系の土台を継続しています。
- 新しいWaterSystem分類により、WaterSystem*.txt/logはWSではなくWATERSYSTEMとして扱われます。
